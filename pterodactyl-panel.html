Pterodactyl Panel

Root folder

sudo -s

Update & Upgrade VPS

apt update && apt upgrade -y && apt install docker-compose -y

Make Required Folder

mkdir pterodactyl && cd pterodactyl && mkdir panel && cd panel

Download Installer file

wget https://github.com/KSWarrior/Make-Own-Hosting/releases/download/PterodactylPanel/docker-compose.yml

Start Installation

docker-compose up -d

Make User For Panel

docker-compose run --rm panel php artisan p:user:make
